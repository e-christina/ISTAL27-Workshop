This repository contains the code to run Average Logarithmic Distance Frequency (henceforth ADFL; Savický & Hlaváčová, 2002). 
The code was created by AI (Gemini Pro and ChatGPT). 
It measures ALDF for British Acadmic British English corpus (hereafter BAWE; Nesi, Hilary; Gardner, Sheena; Thompson, Paul and Wickens, Paul, 2008) words 
and it assigns ALDF values to L2 learner essays.
The output presents:
a) proportion (i.e. how many of the essay words were found in the BAWE),
b) ALDF for content words,
c) ALDF for function words and
d) ALDF for both content and function words. 
The BAWE corpus can be downloaded from the Oxford Text Archive. 
The txt version of BAWE was used. 
L2 Learner essays can be substituted by any (learner) corpus. 
The ALDF task is prepared for the purposes of a workshop so as to present alternatives to GUI NLP tool limitations.
